## Systems

**[BLING](https://github.com/Piyush-072006-B/bling-redteam) — Adversarial Financial Graph Topology Engine**

*Sandboxed research infrastructure for generating and evaluating money-laundering evasion patterns.*

BLING generates synthetic money-laundering graph topologies across 7 structural fraud families — layering chains, round trips, mule networks, structuring bursts, dormant activations, velocity attacks, and fan-in/fan-out mixers. Each topology is mutated through 7 deterministic perturbation strategies (amount jitter, timing spread, node injection, edge weight noise, ghost nodes, cycle variance, account rotation), streamed through Apache Kafka, evaluated against a suite of detection heuristics, and if it evades, exported as a validated evasion pattern for Blue Team review. The Robustness Orchestrator loop runs the full generate → fingerprint → stream → evaluate → mutate → export cycle autonomously — no human input mid-run. Results are immutable: 65+ run records, 57 canonical evasion exports, 0.0% detection rate on all gen-0 exported topologies across all 7 fraud families. The full stack runs as 8 Docker Compose microservices with a React + Vite real-time dashboard connected over WebSocket. No automatic model updates ship without investigator sign-off — this is research infrastructure, not automation of harm.

`Python · FastAPI · Apache Kafka · NetworkX · Docker · Redis · PostgreSQL · React · Vite · WebSocket`

**[MoodNest](https://github.com/Piyush-072006-B/MoodNest-7.2F2) — iOS Mental Wellness Tracker**

*Native iOS mood tracking and pattern visualization.*

MoodNest lets users log daily emotional entries, surface behavioral patterns through native Swift Charts, and build a private record of their mental wellness over time — all on-device, no account required. It's built entirely in SwiftUI using MVVM architecture with Core Data for persistence, not a prototype you stop maintaining after the demo but a real app with a real user experience that holds up under daily use.

`Swift · SwiftUI · iOS · Xcode`
